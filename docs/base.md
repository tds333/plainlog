# Base Types

The ``plainlog._base`` module defines the foundational types used
throughout the library.

## Type Aliases

| Alias | Description |
|-------|-------------|
| `Msg` | Log message content — accepts any type. |
| `Record` | A log record — a plain Python ``dict``. |
